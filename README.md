# UnOredinary Mod

*Minecraft Mod | Version 1.21+ | Forge/Neoforge*

*(note: this mod is still in developpement)*
## Overview
**UnOredinary** is a Minecraft mod that breathes new life into the world of mining by introducing a plethora of unique ores, each with its own special properties and uses. Whether you're a seasoned miner or a curious explorer, UnOredinary offers a fresh and exciting experience for all players.
